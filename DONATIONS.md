## Why support helps
Support can help cover recurring project costs such as:
1. LLM subscriptions/tools used for development workflows (OpenAI, Claude, Copilot)
2. Build/test infrastructure and tooling
3. Documentation and maintenance overhead

## Transparency
Support is intended to help with high-level project costs such as AI tooling, test workflows, hardware experiments, and maintenance overhead.

When major funded expenses become relevant, this file can be updated with periodic notes.

## Non-monetary ways to help now
1. Open issues with reproducible bug reports
2. Submit focused pull requests
3. Improve docs and test coverage
4. Share compatibility test results with DOS/FreeDOS software

Thanks for helping CiukiOS grow over time.
